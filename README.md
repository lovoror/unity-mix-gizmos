# unity-mix-gizmos
提供unity的一些扩展实现，用于技术交流。

## 环境

本项目依赖的环境如下：

| 组件           | 详细信息                                                     | 说明                                                         |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| unity          | 2019.2.9f1                                                   |                                                              |
| Odin Inspector | https://assetstore.unity.com/packages/tools/utilities/odin-inspector-and-serializer-89041 | 非常强大的unity编辑器的序列化扩展插件。不在本仓库中提供，商用项目请支持正版。 |
